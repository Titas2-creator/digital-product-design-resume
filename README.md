# Digital Product Design Resume

A personal, editorial-style resume website built with modern web technologies. The project presents my academic background, skills, projects, achievements, and design interests in a clean, print-friendly A4 layout.

## ✨ Features

- Responsive single-page resume
- Print-friendly A4 design
- Editorial and minimal aesthetic
- Clickable contact links (LinkedIn, Email, Portfolio)
- Easy to customize and update
- Optimized for PDF export

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- HTML5
- CSS3
- Git & GitHub

## 📂 Project Structure

```text
src/
├── components/
├── styles/
├── assets/
└── App.tsx
```

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Titas2-creator/digital-product-design-resume.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## 🎨 Design Inspiration

The resume follows an editorial and publication-inspired design language with:

- Minimal typography
- Structured grid layout
- Print-ready A4 format
- Elegant and professional presentation

## 📄 Exporting as PDF

Open the project in the browser and use:

```text
Ctrl + P → Save as PDF
```

to download the resume as a PDF.
